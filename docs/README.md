# My Wiki

根据 [developer-roadmap](https://github.com/kamranahmedse/developer-roadmap) 给自己立一个 Flag。

使用 [docsify](https://github.com/docsifyjs/docsify) 搭建。

## docsify 常用命令

```shell
npm i docsify-cli -g

docsify init ./docs

docsify serve docs
```

## 提交规范

> TYPE: MESSAGE

TYPE 可选值：

- config：站点配置
- post：增删改文章
- res：增删改资源
